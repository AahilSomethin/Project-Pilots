# Silentink

Silentink is a modern, silent-first learning web app for non-verbal communication.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

- `/` Landing page
- `/dashboard` Dashboard
- `/sign-language` Sign Language module
- `/morse-code` Morse Code module
- `/braille` Braille module

## Documentation

- `docs/README.md` project overview
- `docs/FEATURES.md` feature breakdown
- `docs/UI_GUIDELINES.md` design and motion rules

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React Query
- Framer Motion

## Notes

- UI is intentionally minimal with sharp geometry.
- Animations are subtle and respect reduced-motion preferences.
