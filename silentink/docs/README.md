# Silentink

Silentink is a silent-first learning platform inspired by Duolingo for non-verbal communication skills.

## Vision

- Teach sign language, Morse code, and Braille text communication in one product.
- Keep feedback visual and tactile only (no sound effects).
- Deliver clear, short learning loops with startup-grade polish.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React Query for client state and async data
- Framer Motion for subtle animation

## Core Sections

- Landing page: product story and mode entry
- Dashboard: XP, streak, and mode progress
- Module pages: Sign Language, Morse Code, and Braille

## Product Principles

- Clean layout, sharp geometry, premium dark UI
- Simple component system and predictable information hierarchy
- Motion should guide attention, never distract
